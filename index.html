HTML Tables Cheat Sheet

Written by Laith Abbas(Full-stack Engineer)

This site was created for a Codecademy project

Element Name Description Example of use Extra tips
Table Inserts the table element into the html file Make sure to include table head and table foot sections if needed
Table Rows Inserts a row into your table These rows can hold data
Table Data These tags can hold any type of data between them Each additional data tag counts as another column in the table
Table headings These can hold the headings for your table (usually on the first row) We must add the scope attribute to specify if it is a column or row heading
Colspan or Rowspan These tags can dictate how far a column or row may span, this is an attribute of the table data tag Make sure you provide an integer as a value to the attribute
Table body This is used to store all the rows of data which are in the main section of the table Make sure to exclude the table head and foot from this
Table Head This is used to enclose the column headings Make sure to only include column headings
Table Footer This is used to enclose footing rows This footer section may include a row for sums, diffrences and so on
